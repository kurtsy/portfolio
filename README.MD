### Running Locally
In your terminal in the root of the directory:

```
cd ~/Documents/6_CODE/kurt-duran-yalcin/
dev_appserver.py app.yaml
```

App will be running at localhost:8080.

### Deploying
In your terminal in the root of the directory:

```
cd ~/Documents/6_CODE/kurt-duran-yalcin/
appcfg.py update app.yaml 
```
